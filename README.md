## How it works

Each letter deterministically maps to a note in the pentatonic scale (C, D, E, G, A across octaves). Vowels get warm, sustained sine tones. Consonants get shorter, percussive plucks. Reverb and delay create a sense of space. The result: every name has a unique, beautiful melody.
